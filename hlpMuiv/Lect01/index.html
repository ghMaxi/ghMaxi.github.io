Высокоуровневые методы программирования

Лекция 01: уровни программирования и их реализация в Python

Уровни программирования

НИЗШИЙ

  • Машинный язык
    Строго говоря, не программирование. Поток 0 и 1, составляющий прямые инструкции процессору конкретной архитектуры.
  • Языки сборки
    Первое программирование. Прямые манипуляцию памятью компьютера. Максимум свободы, минимум структуры. Очень тяжело работать.
  • Структурное программирование
    Наделение прямых манипуляций памятью конкретным смыслом. Объединение осмысленных блоков манипуляций в подпрограммы. Составление программы из подпрограмм. Управление подпрограммами с помощью стандартных структурных операторов.
  • Высокоуровневое программирование
    Одновременное выполнение нескольких программ. Управление взаимодействием между программами на уровне архитектуры программного продукта.

ВЫСШИЙ

Особенности Python

Python работает начиная со структурного уровня. Python не имеет прямого доступа к манипуляциям с памятью.

Структурное программирование обеспечивается в Питоне следующими способами:

  • Фиксирование используемых полей памяти в виде переменных
  • Условные и цикловые операторы (if, for, while)
  • Выделение подпрограмм двоеточиями и отступами

Высокоуровневое программирование

обеспечивается в Питоне следующими способами:
  • Выделение программ в методы. Управление "быстрыми" программами путём вызова методов.
  • Выделение "долгоживущих" программ в классы. Управление посредством создания и запуска копий "долгоживущих" программ в виде программных объектов.
  • Создание модулей, позволяющих создавать устойчивые группы методов, классов и полей.
