Алгоритмизация и программирование

Лекция 1:
    Понятие алгоритмизации,
    программы на Python

Дискретизация
    Моделирование феномена реальности
    в виде тесктовой записи

    Формат данных:
        Способ организации данных
        в текст

    Пример "Морской бой":
        Игровое поле
            Размеры
                int size_x
                int int_y
        Типы кораблей
            Название
                string name
            Размер
                int length
            Макс. количество
                int size
        Корабли
            Тип
                ship_type
            Положение
                int x_topLeft
                int y_topLeft
                boolean is_vertical
            Клетки корабля
                bool is_alive
        Игрок
            string name
            list visible_ships
            bool side

Рендеринг
    Технический термин для отрисовки графики
    Преобразование текста в аналоговый сигнал

    Алгоритм отрисовки поля:
        - рисуем задник поля без корабля
        - определяем видимые корабли ???
        - отрисовать промахи
        - отрисовать видимые корабли
        - отрисовать попадания
        - опредить и отрисовать активные поля

Алгоритмизация (в общем):
    Разделение деятельности на шаги

Алгоритмизация (для программирование):
    1) Определение формата и получение исходных
        данных (дискретизация)
    2) Определение способа вывода и требований
        к выводу результата (рендеринг)
    3) Собственно описание шагов преобразования
        текста исходных данных в текст для
        рендеринга

Языки программирования:
    - любой я/п решает пункт 3 алгоритмизация
    - самый фундаментальный язык:
        машинный код
            - не читаем человеком
            - индивидуален для каждой
              процессорной архитектуры
    - ассемблер ("язык сборки")
        - прямые инструкции по манипуляции памяти
        - читаем для специалиста
        - наиболее опасен
    - структурное программирование
        - переменные - выделение в памяти
             места под определённые осмысленные
             блоки данных
             - формат данных
             - адрес в памяти
        - функции
            - устойчивые повторяющиеся инструкции
            - пример:
                   attack(x, y)
        - блоки управления
            - if, for, while
    - Разделение программных парадигм
        - функциональное
        - ООП
        - логическое
