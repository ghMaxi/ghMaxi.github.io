Высокоуровневые методы программирования.
Семинар 1 - планирование сверху-вниз, реализация снизу-вверх

Архив рабочих папок по результатам семинара: Здесь

Самостоятельная работа:
1. Создайте аккаунт на GitHub
2. Скачайте и поставьте на компьютер GitHub Desktop
     2.1 В процессе установки залогиньтесь в аккаунт GitHub.
3. Выберите в команде человека, который создаст пустой репозиторий на GitHub
     3.1 Всем остальным надо будет выдать доступ к этому репозиторию.
     3.2 Это делается в разделе settings -> manage access. Доступ выдаётся по логином GitHub или email
4. Положите .gitignore для WinForms
    в репозиторий и сделайте commit и push на GitHub.
5. Создайте новый проект Windows Forms и положите его в репозиторий.
    5.1 Удостоверьтесь что .gitignore лежит в корне проекта и работает правильно
        (Файлы служебных папок проекта скрыты из Github Desktop)
6. Пусть все участники получат копию проекта через GitHub
7. Приступите к реализации проекта Messenger. Для этого:
    7.1 Определитесь с функционалом Вашего месенджера (с учётом того, что работать он будет в локальной сети)
    7.2 Нарисуйте примерный дизайн Вашего мессенджера
    7.3 Реализуйте в проекте минимальный прототип окна мессенджера:
        7.3.1 Поле ввода сообщения
        7.3.2 Кнопка отправки сообщения
        7.3.3 Место для вывода всех отправленных сообщений
    7.4 Подключите к кнопке функцию переноса сообщения из поля ввода в общий вывод.
8. В качестве результата - сделайте PullRequest последней версии репозитория и присылайте ссылку преподавателю
    8.1 Форма сдачи результатов: ссылка
